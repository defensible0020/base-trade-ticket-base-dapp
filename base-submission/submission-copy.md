# Base Trade Ticket

## App Name

Base Trade Ticket

## Tagline

Match one swap

## Description

Post a simple swap request on Base, let one wallet respond, and accept the match with a clear onchain record.

Character count: 112

## Category

Swap / matching / onchain utility

## Short Pitch

Base Trade Ticket gives people a clear way to post a simple trade intent, find one responder, and mark the match accepted on Base.

## Production Environment Variables

- `NEXT_PUBLIC_TRADE_TICKET_CONTRACT_ADDRESS`: deployed trade ticket contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
