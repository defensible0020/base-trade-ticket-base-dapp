# Base Trade Ticket

### What is this?

A Base dApp for skill trades.

### What does the user do?

They connect a wallet and offering a small exchange.

### What remains?

A trade card, plus public builder proof.

### How do I run it?

```bash
npm install
npm run dev
```

### What is it built with?

Next.js UI plus wagmi/viem for wallet and Base chain behavior.

### What should never be committed?

Do not commit `.env`, private keys, seed phrases, RPC keys, GitHub tokens, or Vercel tokens. Use `.env.example` only for placeholders.
